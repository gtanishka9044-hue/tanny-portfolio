## üêõ Common Issues & Solutions

### Issue: Website doesn't load
**Fix:** 
- Make sure all files are in correct folders
- Check file paths are correct (case-sensitive!)
- Open browser console (F12) to see errors

### Issue: Styles not showing
**Fix:**
- Verify CSS files are in `css/` folder
- Check HTML links to CSS are correct
- Hard refresh browser (Ctrl+Shift+R)

### Issue: JavaScript not working
**Fix:**
- Check JS files are in `js/` folder
- Verify scripts are included before `</body>`
- Check browser console for errors

### Issue: Mobile menu not working
**Fix:**
- Make sure `js/main.js` is loaded
- Check browser console for errors
- Verify mobile menu toggle button exists

### Issue: Tasks not saving
**Fix:**
- Check browser allows Local Storage
- Verify `js/tasks.js` is loaded
- Check browser console for errors
